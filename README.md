# Flutter - Netflix Clone

Netflix Clone App I made with Flutter.


## Run Locally

Clone the project

```bash
  git clone https://github.com/yagizdo/netflix_clone_flutter
```

Go to the project directory

```bash
  cd netflix_clone_flutter
```

Install dependencies

```bash
  flutter pub get
```

Run App

```bash
  flutter run
```
